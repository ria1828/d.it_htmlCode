Responsive Web Design and Freelancing


  1. What do you mean by freelancing?

    2.
    = A freelancer or freelance worker is a self-employed person who earns money by providing services to multiple clients.

  2. What do you mean by Responsive?

    3.
    = Responsive design is an approach to web page creation that makes use of flexible layouts, flexible images and cascading style sheet media queries.
    The goal of responsive design is to build web pages that detect the visitor's screen size and orientation and change the layout accordingly.

  3. What is Web site?

    4.
    = A website is a collection of web pages and related content that is identified by a common domain name and published on at least one web server.
    Examples of notable websites are Google, Facebook, Amazon, and Wikipedia.

  4. What do you mean by web page?

    5.
    = A web page is a hypertext document on the World Wide Web. Web pages are delivered by a web server to the user and displayed in a web browser.
    A website consists of many web pages linked together under a common domain name.

  5. What is web design?

    6.
    = Web design is the process of designing the visual look and feel of a website.
